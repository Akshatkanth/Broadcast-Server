# Broadcast Server
A simple **CLI-based broadcast server** built using **Node.js and WebSockets**.
This project demonstrates how real-time communication works between a server and multiple clients using persistent WebSocket connections.
https://roadmap.sh/projects/broadcast-server
---

## Features
* CLI-based server and client
* Real-time message broadcasting
* Supports multiple connected clients
* Graceful handling of client connect/disconnect
* Lightweight implementation using raw WebSockets (`ws`)

---

## Tech Stack
* Node.js
* WebSockets (`ws`)
* JavaScript (CLI-based)
